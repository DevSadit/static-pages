Competitive Programming Division

The Competitive Programming Division aims to equip members with the skills
to tackle algorithmic challenges and excel in programming contests like ICPC, Codeforces, and LeetCode.

Our Achievements

Levels

Level 1: Beginner (CF Rating: 0 - 800 or equivalent)

For newcomers to competitive programming, this level covers basic programming concepts, data types, loops, and simple problem-solving skills.

Level 2: Intermediate (CF Rating: 800 - 1300 or equivalent)

Designed for programmers with basic knowledge, this level focuses on data structures, algorithms, and medium-level problem-solving techniques.

Level 3: Advanced (CF Rating: 1300 -   or equivalent)

For experienced programmers, this level includes complex algorithms, advanced data structures, dynamic programming, and optimization techniques.
